A text is a sequence of paragraphs.

A segment is a sequence of characters in a paragraph.

 

The trivial segment is the sequence of all characters in the paragraph.

A proper segment is one that is not trivial.

Naturally, the set of all trivial segments identifies with the text.

Such alternate forms of identification help with formulations and implementations. For example, it allows to uniformize, as segment, the type of component of sets possibly containing both paragraphs and proper segments, as in treatments below.

An irregular segment is one that spans more than one paragraph with proper status in the first or the last or both. In the current schema such concept is represented by a set of segments.

 

An annotation is a label associated with entities as follows.

A terminal annotation is associated with a non-null set of segments.

A meta-annotation is associated with a sequence of annotations. (Big issue discussed elsewhere.)
